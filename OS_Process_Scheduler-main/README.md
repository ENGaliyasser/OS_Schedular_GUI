# OS_Process_Scheduler
![image from program](https://github.com/abdelrahman99999/OS_Process_Scheduler/blob/main/screenshots/4.PNG?raw=true)
## Types of schedulers supported:
1. FCFS
2. SJF 
    - Preemptive
    - Non Preemptive
3. Priority 
    - Preemptive
    - Non Preemptive 
4. Round Robin
